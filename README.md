# polydmon
watch polyswarm go

## install

```
python setup.py install
```

## usage

```
$ polydmon --help
Usage: polydmon [OPTIONS]

Options:
  -e, --event [bounty|assertion|vote|all]
  -u, --uri [wss://nu.k.polyswarm.network/v1/events/?chain=side|wss://lima.polyswarm.network/events?chain=side|all]
  -j, --json
  --help                          Show this message and exit.
```
